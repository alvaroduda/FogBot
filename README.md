# FogBot 🤖

Bot oficial de gerenciamento e vendas para o servidor **Fog Client**.

Este bot foi desenvolvido para automatizar o processo de vendas, suporte e gerenciamento de membros no servidor do Discord. Ele é otimizado para rodar com baixo consumo de memória (compatível com planos gratuitos como Discloud).

## 🚀 Funcionalidades Principais

### 🎫 Sistema de Vendas e Tickets
- **Criação de Tickets**: Canais privados criados automaticamente para cada cliente.
- **Botões Interativos**: Interface amigável para abrir tickets, fechar e confirmar pagamentos.
- **Detecção de Comprovantes**: Identifica automaticamente quando um usuário envia uma imagem ou PDF no ticket e notifica a staff.
- **Painel de Administração**: Botões exclusivos para staff gerenciar o encerramento de tickets.

### 🛡️ Gerenciamento Automático
- **Auto-Role (Membro)**: Atribui automaticamente o cargo "Membro" para novos usuários que entram no servidor.
- **Setup Automático (`/setup`)**: Cria e configura todos os cargos, canais e permissões necessários com um único comando.
- **Logs de Vendas**: Registra todas as transações finalizadas e tickets fechados no canal de logs.

### ⚡ Performance e Estabilidade
- **Baixo Consumo de RAM**: Otimizado para rodar com menos de 100MB.
- **Tratamento de Erros**: Sistema robusto que evita quedas por erros inesperados.
- **Reinício Automático**: Configurado para reiniciar em caso de falhas críticas.
